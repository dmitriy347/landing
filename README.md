# Лендинг-портфолио — Дмитрий Фоменко

Одностраничный лендинг-портфолио, подготовленный для тестового задания на позицию
**AI Generalist / AI Automation Specialist** (Goulash.tech).

Сайт фокусируется не на перечислении опыта, а на том, почему этот опыт релевантен
роли: главный кейс — `tool-issuance-bot`, AI как звено в реальном рабочем пайплайне,
а не как чат-интерфейс.

Статический сайт без зависимостей — достаточно открыть `index.html` в браузере,
либо поднять локальный сервер:

```bash
npx serve .
```
